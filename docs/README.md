# Журнал выполнения практики

## Этап 0: Планирование и самообучение 

### Планирование

В ходе планирования было решено разместить репозиторий на платформе GitHub (https://github.com/TrueNegod/laughing-adventure) в силу того, что все участники команды были знакомы с данной платформой.

Сайт было решено писать на чистом CSS + HTML, с целями: 

- Иметь максимальный контроль над дизайном финального сайта. 
- Изучить и отработать навыки работы с HTML и CSS 

Были намечены основные темы для самообучения: 

- Работа с Git.

Несмотря на общую осведомленность о базовых принципах разработки проектов с использованием Git, необходимо было изучить основную философию позади этой системы, основные способы работы с ней и наладить разработку проекта учитывая полученные знания. 

- Работа с CSS + HTML 

Нужно было изучить данный стек технологий не только для написания сайта, выполнения практики, но и для личного профессионального роста каждого участника

### Самообучение

В течение подготовительного периода (01.06.2026 - 07.06.2026) были изучены темы работы с Git: основные понятия и идеи этой философии, основные команды взаимодействия с удаленным репозиторием, основы планирования разработки и CSS + HTML: основные понятия, создание структуры сайта, ссылок, настройка стилей, наполнение материалами.
